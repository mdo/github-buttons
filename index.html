Introducing the unofficial

GitHub Buttons

Showcase your GitHub repo's success with these two simple, static buttons featuring dynamic watch and fork counts.

Watch button

  • Real-time watchers count
  • Static link to any public GitHub repo
  • Sized as similar share buttons

Fork Button

  • Real-time forks count
  • Static link to any public GitHub repo
  • Sized as similar share buttons
